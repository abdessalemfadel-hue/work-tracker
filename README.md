# WorkTracker

Private, offline Android work-hours tracker with clock in/out, break tracking, notes, history, CSV export, and GitHub Actions APK builds.

## Build

Push to `main` or run **Actions → Build Android APK → Run workflow**. Download the `WorkTracker-debug-apk` artifact after the workflow finishes.

Requirements: Android SDK 34, JDK 17, Gradle 8.2.1.
